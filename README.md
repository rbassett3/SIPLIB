# SIPLIB: A Stochastic Integer Programming Test Problem Library (by Shabbir Ahmed) 

This repository contains a clone of SIPLIB (A Stochastic Integer Programming Test Problem Library) from Prof. Shabbir Ahmed's website [isye.gatech.edu/~sahmed/siplib/](https://www2.isye.gatech.edu/~sahmed/siplib/). In early 2026 Georgia Tech took down this site. Many researchers expressed concerns about the removal of this library of test problems, and as a result the site was brought back online. As of March 10, 2026 the site is back online. This repository serves as a backup for this important resource in case of future disruption.

This repository is a clone of the original website, conducted via
```
wget -r -np -k -p https://www2.isye.gatech.edu/~sahmed/siplib/
```
Navigation is awkward because github doesn't display html by default, but cloning and navigating the files locally via a web browser will work. I've erred on the side of preservation by not making any modifications to the original website.

### Relevant information

- **Citation**. To cite the test problems in this collection either cite the original papers where the problems are described or use the following citation:

```
S. Ahmed, R. Garcia, N. Kong, L. Ntaimo, G. Parija, F. Qiu, S. Sen. SIPLIB: A Stochastic Integer Programming Test Problem Library. https://www2.isye.gatech.edu/~sahmed/siplib, 2015. 
```

- **More Information on Prof. Ahmed**. Prof. Ahmed passed away on June 19, 2019. An "In Memoriam" page can be accessed [here](https://www.isye.gatech.edu/news/memoriam-shabbir-ahmed) (or [here](https://web.archive.org/web/20250212211934/https://www.isye.gatech.edu/news/memoriam-shabbir-ahmed) via Internet Archive if the original is ever deleted).
